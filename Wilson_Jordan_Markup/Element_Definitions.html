A list of words and their definitions

Headings
A heading is what goes at the top of a web page. < h1 > < /h1 >... < h6 > < /h6 >
Paragraphs
A group of sentences that form one idea. < p > < /p >
Bold & Italic
Adds emphasis to words in a paragraph. < b > < /b > for bold and < i > < /i > for italics
Superscript & Subscript
Superscript goes slightly above a number and is used with numural powers < sup > < /sup >. Subscript goes slightly below a number and is used in chemical make ups < sub > < /sub >.
White Space
Is the space(s) between words. Web browsers will collaps all white space so it looks like there is only one space
Line Breaks
Will take your text to the next line < br / >.
Horizontal Rules
It will draw a horizontal line across your screen < hr / >.
Visual Editors
Looks like a word processor but is optimized for coding
Semantic Markup
Adds extra information to your web page. < blockquote > adds quotation marks, < em > adds emphasis to a word
Strong Emphasis
Makes a word bold < strong > < /strong >.
Quotations
Adds quotation marks around some text < q > < /q >.
Abbreviations
A simplified spelling of a word < abbr title="Professor" < Prof > </abbr >.
Acronyms
Usually the first letter of each word to make a new meaning < acronym title="National Aeronautics and Space Administration" < NASA > </acronym >.
Citations & Definitions
Citations is where you get some information < cite > < dfn >.
Author Details
Gives details of a person < address >.
Changes to Content
You have a document, you make a change to that document. < ins >, < del > and < s >
Ordered Lists
Gives a numbered list < ol > < /ol > and < li > < /li >.
Unordered Lists
Gives a bulleted list < ul > < /ul > and < li > < /li >.
Definition Lists
Gives structure a word with a definition < dl > < /dl > starts it, < dt > is for the word and < dd > is for the definition.
Nested Lists
Is a list within a list
